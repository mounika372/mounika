• The Ectron ECT-KRIA-EVK is built around the powerful kria™ SOM from AMD. The Kria™ is a Zynq™ UltraScale+™ MPSoC with quad core 64-bit ARM processor coupled with a large FPGA. The industrial version of the compact SOM has an operation range: -40°C to +100°C for rugged applications in harsh environments.
  • The ECT-KRIA-EVK is the electronics that goes into Ectron’s LG/EG class computers configured as a light gateway and embedded computers. They run Ubuntu Linux and come pre-loaded with the software to build multiple applications. This is one of the most powerful development kits on the market for edge computing applications. A BIST is provided with each board and a simple reference design for use.
